Project Description
This project analyzes open source software from both technical and philosophical perspectives. It includes the study of the origin, licensing, Linux footprint, ecosystem, and comparison with proprietary software. In addition, five shell scripts are implemented to demonstrate practical understanding of Linux and shell scripting concepts.

Shell Scripts Description

Script 1 — System Identity Report
This script displays system information such as Linux distribution, kernel version, current user, home directory, uptime, and date/time.

Script 2 — FOSS Package Inspector
This script checks whether a given open-source package is installed on the system, displays its version, and prints a short description using a case statement.

Script 3 — Disk and Permission Auditor
This script audits important system directories and displays their permissions, ownership details, and disk usage.

Script 4 — Log File Analyzer
This script reads a log file and counts the number of occurrences of a specific keyword. It also displays the last five matching lines.

Script 5 — Open Source Manifesto Generator
This script takes user input and generates a personalized open-source manifesto, saving it to a text file.
