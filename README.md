Project Description
This project explores open source philosophy, licensing, Linux environment, and includes five shell scripts demonstrating key concepts.

Shell Scripts

Script 1 — System Identity Report
Displays system details like kernel, user, uptime, and OS.

Script 2 — FOSS Package Inspector
Checks if a package is installed and shows version and description.

Script 3 — Disk and Permission Auditor
Checks directory permissions and disk usage.

Script 4 — Log File Analyzer
Counts keyword occurrences in a log file.

Script 5 — Open Source Manifesto Generator
Creates a personalized manifesto based on user input.

How to Run

1. Open terminal
2. Give permission:
   chmod +x script_name.sh
3. Run:
   ./script_name.sh

Example:
./script4.sh sample.log error
